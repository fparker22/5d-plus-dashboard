# 5D+ Evaluation Dashboard

A premium, agentic dashboard for managing 5D+ teacher evaluations. Built with Next.js and Glassmorphism design.

## Features

- **Dashboard**: Quick stats and recent activity.
- **Teacher Directory**: Manage roster and start evaluations.
- **Evaluation Wizard**: 
  - **Upload**: Drag & drop transcripts.
  - **Coding**: Interactive split-screen interface for applying 5D+ dimensions.
  - **Analysis**: Structured input for Noticings, Wonderings, Glows, and Grows.
  - **Review**: Generate PDF/Email artifacts.
- **Data**: All data is stored locally in `data/` for privacy and portability.

## getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Open in Browser**:
    Navigate to [http://localhost:3000](http://localhost:3000).

## Customization

- **Styles**: Edit `app/globals.css` to tweak the Glassmorphism theme variables.
- **Data**: Update `data/teachers.json` to manage your roster.
